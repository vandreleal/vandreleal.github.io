Description

Oxidizer is an asynchronous Rust ORM based on tokio-postgres and refinery. Two powerful libraries that give performance and reliability to perform database interactions.

A Rust ORM based on tokio-postgres and refinery that helps you reduce the boilerplate of writing entities, tables and migrations when using tokio-postgres and refinery.

  • Asynchronous from the ground up. All the database operations are efficiently handled by tokio at runtime.
  • Oxidizer macros generate code to access relations between entities with ease. Forward and reverse relations are supported.
